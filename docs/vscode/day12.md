# debug

## start
- f5

## stop
- shift + f5

## restart
- shift + cmd + 0
修改`workbench.action.debug.restart`键值

## 打断点
- shift + cmd + 9
修改`workbench.action.debug.toggleBreakpoint`键值

## step over
- cmd + '
修改`workbench.action.debug.setpover`键值

## step into
- shift + cmd + ' 
修改`workbench.action.debug.stepinto`键值

## step out
- f5

## 也可配合vspacecode插件使用
